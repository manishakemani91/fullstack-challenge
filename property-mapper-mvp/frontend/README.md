<h1 align="center"> 
Property Mapper Frontend
</h1>

## Dependencies 
- [nodejs ^14.18.0](https://nodejs.org/)



## Install

```
npm install
```
## Create .env file for environment variables 
create **.env** file in root folder .env
```
VITE_API_URL= url
```

## Run / Develop

```
npm run dev
```

## Build

```
npm run build
```

## Preview

```
npm run preview
```

## Lint

```
npm run lint

```
